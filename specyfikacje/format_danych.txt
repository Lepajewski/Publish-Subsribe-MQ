kolejność danych w przesyłanych danych:
1. nr sygnału (enum sygnałów z brokera)
    sygnały:

    - connack od klienta
        broker czekuje czy już istnieje połączenie od danego ip:port
    - connack od brokera
        2. wygenerowane id klienta                                  (int)

    - ping od brokera i klienta
        pusta wiadomość budząca

    - suback od klienta
        2. długość nazwy topicu                                     (size_t)
        3. nazwa topicu
    - suback od brokera
        2. kod powodzenia subskrybcji (jakieś 0 lub 1 czy coś)      (suback_success_code)
        ----- tylko jeśli 2. nie jest SUBACK_FAILURE
        3. długość nazwy topicu                                     (size_t)
        4. nazwa topicu
        5. ilość wiadomości w topicu                                (size_t)
        dla każdej wiadomości w topicu {
            6. id wiadomości                                        (int)
            7. długość wiadomości                                   (size_t)
            8. wiadomość
        }

    - puback od brokera
        2. id wiadomości                                            (int)
    - puback od klienta
        2. długość nazwy topicu                                     (size_t)
        4. nazwa topicu
        5. długość wiadomości                                       (size_t)
        7. wiadomość
    
    - newmes od brokera
        2. długość nazwy topicu                                     (size_t)
        3. nazwa topicu
        4. id wiadomości                                            (int)
        5. długość wiadomości                                       (size_t)
        6. wiadomość

    - unsuback od klienta
        2. długość nazwy topicu                                     (size_t)
        4. nazwa topicu
    - unsuback od brokera
        2. kod powodzenia desubskrybcji (jakieś 0 lub 1 czy coś)    (int)

    - disconnack od klienta
        pusty sygnał
    - disconnack od brokera
        pusty sygnał
